# Security Policy

## Reporting a Vulnerability
- Открыть Security advisory (Security → Advisories → New draft advisory)
- Или отправить письмо: security@example.com  
- SLA: ответ в течение 3 рабочих дней, фиксы для high/critical — до 7 дней.

## Supported Versions
- Текущая ветка main (активная)
- Последний релиз (tag)
